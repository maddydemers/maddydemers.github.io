Interactive Acitvity Map

Personal Project

Click on a marker to zoom in and view information about an activity. Return to original zoom state.

Thanks to the generous support of Strava's API, I was able to compile my extensive personal activity upload history on the platform, totaling over 1,000 activities. Leveraging integrations with Leaflet and OpenStreetMap, I have constructed a visualization showcasing all the locations worldwide where I have logged a Strava activity*. This project epitomizes my enthusiasm for both movement and travel. I eagerly anticipate updating the map with my ventures to new destinations in 2025! Please visit my GitHub to see the code I developed to build this map.

*For privacy reasons, I decided against real-time updates during the initial design process.
